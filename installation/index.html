Guida sull'installazione

1. Installazione locale (Windows + XAMPP)

Questa è la modalità consigliata per sviluppare e testare Didattix in locale.

1.1 Requisiti

  • Windows 10/11
  • XAMPP, un ambiente di sviluppo PHP (Apache + MariaDB + PHP + Perl)

1.2 Installazione di XAMPP

  • Scarica XAMPP dal link fornito ai requisiti.
  • Installarlo scegliendo la versione con PHP 8.x.
  • Avvia il pannello di controllo XAMPP.
  • Avviare i moduli Apache e MySQL.

1.3 Posizionamento del progetto

  • Scarica il codice sorgete del progetto dal repository ufficiale
  • XAMPP espone i contenuti web tramite la directory dedicata C:\xampp\htdocs\, crea una directory Didattix all'interno di essa ed estrai il codice sorgente scaricato.

1.4 Database

  • Accedi a phpMyAdmin all'indirizzo http://localhost/phpmyadmin.
  • Crea un nuovo database, il nome è a scelta.
  • Modifica i dati di accesso al database in config.php, lo puoi trovare facilmente in C:\xampp\htdocs\Didattix\private\php\ e modificare l'array database.
  • La root del progetto sarà raggiungibile in locale all'indirizzo http://localhost/didattix/

1.5 Installazione di Composer e PHPMailer

  • Scarica e installa Composer all'indirizzo https://getcomposer.org/
  • Se la directory C:\xampp\htdocs\Didattix\private\php\mailer\ non è presente, crearla manualmente, in seguito aprire il prompt dei comandi in quella cartella.
  • Esegui il comando composer require phpmailer/phpmailer e attendere il processo di installazione. Si noti che la cartella vendor e lo script vendor/autoload.php sono generati da Composer e non fanno parte di PHPMailer.

NOTA: se non si vuole usare Composer, fare riferimento alla documentazione di installazione di PHPMailer su https://github.com/PHPMailer/PHPMailer

1.6 Genera password per app

Per il funzionamento di PHPMailer è necessario aver creato una password per l'app di produzione:

  • Vai su https://myaccount.google.com/security e attiva l'autenticazione a due fattori (2FA).
  • Cerca la voce "Password per app" o "Accedere con le password per le app", se non la trovi basta cercarla nella barra di ricerca.
  • Su "Nome dell'app" digitare il nome dell'app desiderato, nel nostro caso "Didattix" e cliccare su "Crea".
  • Verrà generato un codice a 16 cifre, copialo e incollalo in config.php nell'array 'mail' => [ 'password' => 'your_password' ], assicurati di modificare anche l'username e gli altri dati, se sono necessari.

NOTA: la logica delle email è già gestita nel codice sorgente, non dovrai fare altro che ritornare al config.php e modificare l'array mail.

Il sistema è pronto per l'esecuzione.

2. Installazione in ambiente di produzione (Linux server)

Per un deployment reale (VPS, server dedicato, cloud), la configurazione tipica avviene su distribuzioni Linux come Ubuntu/Debian.

2.1 Requisiti

  • Ubuntu 20.04+ o Debian 11+
  • Web server (Apache o Nginx)
  • PHP 8.x + estensioni
  • MySQL/MariaDB
  • Composer (opzionale)

2.2 Installazione dei pacchetti

Installa i pacchetti necessari per l'esecuzione di Didattix:

sudo apt update
sudo apt install apache2
sudo apt install php php-mysql php-mbstring php-json php-openssl php-fileinfo
sudo apt install mariadb-server

2.3 Installazione del progetto

  • Scarica il codice sorgete del progetto dal repository ufficiale
  • Estrai i file del codice sorgente e spostali nella directory /var/www/didattix/.

Controllare la documentazione relativa all'installazione del web server e del database nelle pagine ufficiali, una guida garantita puoi trovarla in:

  • https://www.digitalocean.com/community/tutorials/how-to-install-the-apache-web-server-on-ubuntu-20-04 per il web server
  • https://www.digitalocean.com/community/tutorials/how-to-install-mysql-on-ubuntu-20-04 per il database

2.4 Installazione di Composer e PHPMailer (opzionale)

Puoi installare facilmente composer eseguendo:

sudo apt install composer
cd /var/www/didattix
composer install

Per installare PHPMailer, scorrere in alto alla sezione 1.5 (Installazione di Composer e PHPMailer)
